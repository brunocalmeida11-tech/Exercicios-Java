# Exercícios de Algoritmos em JavaScript (Node.js)

Repositório contendo a resolução dos exercícios desenvolvidos em JavaScript utilizando Node.js.

## 📚 Conteúdo

Os exercícios estão organizados em três etapas:

```text
01_EstruturaSequencial/
02_EstruturaDeDecisao/
03_EstruturaDeRepeticao/
```

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- Node.js
- npm
- VS Code
- Git/GitHub
- Ubuntu Linux

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/brunocalmeida11-tech/Exercicios-Java.git
```

Entre na pasta do projeto:

```bash
cd Exercicios-Java
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Executando os Exercícios

Exemplo:

```bash
node 01_EstruturaSequencial/ex01.js
```

ou

```bash
node 02_EstruturaDeDecisao/ex05.js
```

---

## 📥 Dependência Utilizada
